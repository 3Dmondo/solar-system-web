models, including polygon data in any format. The guidance says NASA content is
generally not subject to U.S. copyright, can be used for computer graphical
simulations and Internet web pages, should acknowledge NASA as the source, and
must not imply NASA endorsement. The Phobos and Deimos model pages do not mark
the files as third-party copyrighted.

Milestone 15 runtime tier:
Use optimized GLB files served from public/meshes/ for the first runtime pass.
Do not use Draco or Meshopt in the first pass; avoiding decoder assets keeps the
static GitHub Pages build simpler while the orientation, scale, picking, and
bundle-size path is still being proven. Review Draco or Meshopt only if the
production bundle/static asset review shows the plain optimized GLB tier is too
large.

Conversion commands:
npx --yes @gltf-transform/cli optimize <scratch>/phobos-source.glb public/meshes/phobos.glb --compress false
npx --yes @gltf-transform/cli optimize <scratch>/deimos-source.glb public/meshes/deimos.glb --compress false
npx --yes @gltf-transform/cli resize public/meshes/phobos.glb public/meshes/phobos-smalltex.glb --width 16 --height 16
npx --yes @gltf-transform/cli resize public/meshes/deimos.glb public/meshes/deimos-smalltex.glb --width 16 --height 16
Move the `*-smalltex.glb` outputs back to the runtime filenames after checking
the size delta.
